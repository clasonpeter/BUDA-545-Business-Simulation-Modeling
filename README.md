# BUDA-545-Business-Simulation-Modeling

Spring 2026 | West Virginia University

This course introduces computer-based simulation for business decision-making. Students design, build, and analyze simulations using AnyLogic and Excel to model real-world business systems.
Modules

- Module 1 – Introduction to simulation and Monte Carlo methods
- Module 2 – Conceptual modeling and AnyLogic basics
- Module 3 – Discrete-event modeling and single-server queuing (1)
- Module 4 – Single-server queuing (2), schedules, and multi-server queuing
- Module 5 – Model testing, experimentation, and output analysis
- Module 6 – Agent-based modeling and simulation
- Module 7 – System dynamics and hybrid simulation
